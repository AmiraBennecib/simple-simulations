# Computational Physics Portfolio

## Project 1: 1D DFT from Scratch
Minimal implementation of Density Functional Theory in 1D using NumPy.

**Physics covered:**
- Finite difference solution of Schrödinger equation
- Self-Consistent Field (SCF) iteration
- Hartree potential via Poisson equation  
- Exchange-correlation via LDA approximation

**Verified against:** analytical solution En = n²π²/2L²
